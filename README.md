# docto
Terminal for pharmacy with the aim of finding its drugs when you are travelling in another country.

## Run the project
### Run the backend API
```sh
42sh$ cd src/
42sh$ node doctoAPI.js PORT LOCATION
```

### Run the frontend website
```sh
42sh$ cd website/
42sh$ npm start
```
